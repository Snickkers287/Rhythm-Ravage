# ⚡ Rhythm Ravage

A high-octane, monster-themed rhythm game built with the Godot Engine. Face the beast and hit the notes to the beat of 16-bit industrial metal.

![Rhythm Ravage Cover](https://your-image-link-here.com) 

## 🎮 Overview
**Rhythm Ravage** is a fast-paced rhythm experience where players must time their inputs to falling notes across four lanes. Featuring custom high-performance rendering and dynamic "game juice," it's designed to provide a crunchy, responsive arcade feel.

## 🚀 Key Features
- **Custom Render Pipeline:** Built using Godot's `_draw()` system for maximum performance and a unique visual style.
- **Dynamic Feedback:** Rhythmic screen shake, lane flashes, and "bounce" animations that respond to your performance.
- **Combo System:** Score multipliers that reward accuracy and long streaks.
- **End-Game Analytics:** Detailed stats screen showing Final Score, Max Combo, and percentage-based Accuracy.
- **Audio Sync:** Precise beat-tracking logic to ensure gameplay stays perfectly in time with the music.

## 🛠️ Technical Breakdown
This project was developed as a 5-hour "speed-run" challenge, focusing on game feel and optimization:
* **Hour 1-2:** Core engine setup and audio-beat synchronization.
* **Hour 3:** State management, health systems, and core gameplay loop.
* **Hour 4-5:** Refactored UI from standard nodes to a custom-draw system; implemented "Juice" (lerp-based animations and screen shake).

## 🎹 Controls
- **Lane 1:** `1`
- **Lane 2:** `2`
- **Lane 3:** `3`
- **Lane 4:** `4`
- **Restart:** `R`

## 📦 Installation
1. Clone this repository.
2. Open the project in **Godot 4.x**.
3. Press `F5` to play!
